## 有機農業とは

国際標準として定義されている有機農業の概念は以下の通り。

> 有機農業は、生物の多様性、生物的循環及び土壌の生物活性等、農業生態系の健全性を促進し強化する全体的な生産管理システムである

日本の法律ではより具体的な要件が定められている。

> ｢有機農業」とは、化学的に合成された肥料及び農薬を使用しないこと並びに遺伝子組換え技術を利用しないことを基本として、農業生産に由来する環境への負荷をできる限り低減した農業生産の方法を用いて行われる農業

つまるところ、有機農業とは、

1. **化学肥料や農薬や遺伝子組換え技術を使わず**
2. **生物循環や土壌の生物活性を上手く活用**し
3. **生物多様性や農業生態系への負荷を低減/健全性を促進する**

農業のことを指す。

## 世界的なトレンド

市場・作付面積ともに右肩上がりのトレンド。
SDGS や環境問題への意識の高まりから、特にヨーロッパでニーズが高まっている。

### グローバル市場規模

世界における有機農業の規模は右肩上がりで増加している。

- **市場規模**
  - 2000-2020 年の 20 年間で、世界の有機農業市場は約 8 倍に成長
  - 2020 年の市場規模は約 1.2 兆ドル

マーケットとしては、北アメリカ(アメリカ)が最も大きく、次いでヨーロッパ(ドイツ、フランス)、アジア(中国)の順。

| 地域       | 売上規模（十億ユーロ） |
| ---------- | ---------------------- |
| 北アメリカ | 53.7                   |
| ヨーロッパ | 52.0                   |
| アジア     | 12.5                   |
| オセアニア | 1.6                    |
| 南アメリカ | 0.8                    |
| アフリカ   | 0.02                   |

一人あたりで換算すると、以下のように上位はヨーロッパ諸国が占めている。

### 栽培面積

栽培面積の規模も右肩上がりで増加している。

- **栽培面積**
  - 1999-2020 年の 20 年間で、世界の有機農業の栽培面積は約 7 倍に成長
  - 2020 年の栽培面積は約 7,490 万 ha
  - 世界の農地面積の約 1.6%

地域別に見ると、オセアニア(オーストラリア)が最も広く、次いで欧州、南米(アルゼンチン、ウルグアイ)の順。

| 地域       | 有機栽培面積 (百万 ha) |
| ---------- | ---------------------- |
| オセアニア | 35.9                   |
| ヨーロッパ | 17.1                   |
| 南アメリカ | 9.9                    |
| アジア     | 6.1                    |
| 北アメリカ | 3.7                    |
| アフリカ   | 2.1                    |

地目割合で見ると圧倒的に永年草地・牧草地が多い。

| 地目                       | 面積（百万 ha） | 割合 |
| -------------------------- | --------------- | ---- |
| 永年草地・放牧地           | 49.5            | 66%  |
| 耕地                       | 13.6            | 18%  |
| 永年作物地                 | 5.2             | 7%   |
| その他（野生採集・養蜂等） | 6.6             | 9%   |

## 日本の傾向

### 国内市場と栽培面積

日本でも同様に右肩上がりのトレンドではあるが、他国と比較すると市場規模はかなり小さい。

- **市場規模**
  - 2009-2017 年の 8 年間で、国内の有機農業市場は約 1.4 倍に成長
  - 2017 年の市場規模は約 1,850 億円

国土の問題も有るが、栽培面積も他国と比較するとかなり小さい。

- **栽培面積**
  - 2010-2020 年の 10 年間で、国内の有機農業の栽培面積は約 1.5 倍に成長
  - 2020 年の栽培面積は約 2.5 万 ha
  - 全農地面積の約 0.6%

地目ごとだと、普通畑が最も多く、次いで牧草地、水田の順。
特に牧草地は近年増加傾向にある。
(地目別面積は有機 JAS のデータ)

| 地目   | 面積（千 ha） | 割合 |
| ------ | ------------- | ---- |
| 普通畑 | 5.6           | 40%  |
| 牧草地 | 3.4           | 24%  |
| 水田   | 3.0           | 21%  |
| 茶畑   | 1.4           | 10%  |
| 樹園地 | 0.56          | 4%   |
| その他 | 0.16          | 1%   |

農産物についてより細かく見ると、野菜が最も多く、次いで米、緑茶、果樹の順。

| 区分 | 生産量 | 有機 JAS 割合(%) |

### 生産者の実態

2009-2020 年の 11 年間で、国内の有機農業の生産者数はほぼ横ばい。
一方で新規参入者は有機農業を選択する割合は少なくない。

- **生産者数**
  - 2020 年の生産者数は約 3,790 戸
  - 新規参入者のうち、有機農業を選択する割合は約 16.9%(一部作物のみも含めると 22.8%)

有機農産物の JAS 認証事業者を都道府県別に見ると、鹿児島県が最も多く、次いで北海道、熊本県の順。
全体では 5,678 事業者が有機 JAS 認証を取得している。
なお、鳥取は 37 位で 32 事業者のみ。

| 順位 | 都道府県 | 有機農産物の経営体数 |
| ---- | -------- | -------------------: |
| 1    | 鹿児島県 |                  339 |
| 2    | 北海道   |                  332 |
| 3    | 熊本県   |                  209 |
| 4    | 新潟県   |                  175 |

### 生産者の意識

有機農業を選択した理由としては、以下のような意識がある。

- よりよい農産物を提供したい (69.2%)
- 農薬・肥料などのコスト低減 (35.5%)
- 環境負荷を少なくしたい (32.6%)

一方、有機農業を拡大するかどうかについては、**7 割以上の生産者が現状維持**と考えている。
特に、理由としては、以下のように**人手不足やコストの問題**が挙げられている。

- 人手が足りない (47.2%)
- 維持管理に手間がかかる (45.5%)
- 資材コストが高い (22.6%)
- 収益が上がらない (20.2%)

## 政策

農林水産省が定めた「みどりの食料システム戦略」によると、2050 年までに**有機農業の割合を 25%(100 万 ha)に拡大**にすることを目指している。

戦略としては、以下のように段階的に進めていく。
2040 年の段階までに次世代型有機農業技術を確立させ、そこから 10 年で指数関数的に拡大させるイメージ。
並行して国民への啓蒙活動や輸出促進も行う。

| 年代      | 戦略                                                                     | 目標面積(万 ha) |
| --------- | ------------------------------------------------------------------------ | --------------- |
| 2020-2030 | 現場の実践技術の体系化と普及<br/>国民運動・輸出促進                      | 6.3             |
| 2030-2040 | スマート技術等による次世代有機農業技術の開発・確立<br />国内外の需要拡大 | 30              |
| 2040-2050 | 化学農薬等に依存しない栽培技術の一般化                                   | 100             |

具体的な施策の方向性は以下の通り。

- **人材育成**
- **産地づくり**
- **販売機会の多様化**
- **消費者の理解の増進**
- **技術開発・調査**

特に、消費者への働きかけによる市場拡大と労働人口の確保が重要視されている。

## まとめと所感

有機農業は環境負荷を低減するための重要な手段であり、世界的に需要が高まっている。
日本でも市場は拡大しているが、他国と比較するとまだまだ小さい。
特に、日本においては、生産者側のコストや人手不足の問題が大きく国内市場も小さいため、拡大には時間がかかると考えられる。

また、国の施策によると、イノベーションを通じた次世代型有機農業技術の確立を目指している。
特に、生産者側のコストや人手不足の問題を解決するためのスマート農業技術が求められているようだ。

これから IoT や AI について学びを深めていく中で、そういった次世代型有機農業技術として何ができるのかを考えていきたいと思います。
